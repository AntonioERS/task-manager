SELECTORES CSS

Selectores

Selectores de tipo:

También llamados selectores de nombre de etiqueta o selector de elemento:

span {
   background-color:yellow;
}

El selector universal

El selector universal se indica con un asterisco * y selecciona todos los elementos del documento (o del elemento padre, si está ligado a otro elemento y a un indicador de selección descendiente, por ejemplo):

* {
   margin: 0;
}

Otro uso de selector universal es para facilitar la legibilidad de tus selectores. Por ejemplo: article :first-child seleccionará el primer elemento hijo de un elemento article pero se puede confundir fácilmente con article:firstchild que selecciona cualquier elemento article que sea el primer hijo de otro elemento. Para evitar esta confusión, en el primer caso podemos añadir el selector universal, facilitando la legibilidad: article *:first-child

Selectores de clase

El selector de clase comienza por un punto . y selecciona todos los elementos del documento que estén afectados por esa clase:

.nombreDeLaClase {
   background-color: yellow;
}

Delimitación de clases en elementos particulares

El selector de clase se puede utilizar en conjunción con un elemento en particular para seleccionar los elementos concretos que estén afectados por una clase. Por ejemplo:

h1.nombreDeLaClase {
   background-color: yellow;
}

seleccionará todos los elementos h1 que estén afectados por esa clase.

Delimitar elementos que estén afectados por más de una clase

Los elementos pueden tener más de una clase, pudiendo seleccionar cada clase en particular o elementos que contengan varias o todas las clases:

.nota {
   border: 4px solid #666;
   padding: .5em;
}

.nota.advertencia {
   border-color: orange;
   font-weight: bold;
}

.nota.peligro {
   border-color: red;
   font-weight: bold;
}

Selectores de ID

El selector de ID comienza con almohadilla # y se aplica como el selector de clase, con la diferencia que cada ID solamente se puede utilizar una vez en el documento y a cada elemento sólo se le puede aplicar un ID:

#one {
   background-color:yellow;
}

Se puede puede seleccionar un elemento que tenga la propiedad ID y ese ID puede ir precedido de un selector de tipo que seleccionará el elemento sólo si el ID y el tipo coinciden:

h1#one {
   color: rebeccapurple;
}

El ID tiene una especifidad muy alta y anula los demás selectores. Es por ello que es recomendable utilizar clases en su lugar, pero si es necesario el ID puede utilizarse sin problema.

Selectores de atributo

En HTML los elementos pueden tener atributos que proporcionan un nivel de detalle mayor sobre el elemento que delimitan. En CSS se pueden utilizar selectores de atributo para seleccionar elementos definidos con unos atributos determinados.

Selectores de presencia y valor

Permiten seleccionar elementos a partir de la presencia de un atributo (por ejemplo href) o a partir de varias coincidencias diferentes con respecto al valor del atributo:

[attr]
Se aplica a elementos con el mismo atributo "attr" cuyo nombre es el valor entre corchetes:
a[title]

[attr="value"]
Se aplica a elementos con un atributo "attr" cuyo valor es exactamente "value" (el valor entre comillas). Si el atributo tiene más valores además de exactamente "value" entonces la condición no se cumplirá:
a[href="https://example.com"]

[attr~="value"]
Se aplica a elementos con un atributo "attr" cuyo valor es exactamente "value" (el valor entre comillas) o contiene una lista de valores separados por espacios entre los que se encuentra "value":
p[class~="special"]

[attr|="value"]
Se aplica a elementos con un atributo "attr" cuyo valor es exactamente "value" (el valor entre comillas) o empieza por "value" seguido de un guión:
div[lang|="special"]

Selectores de subcadenas

Permiten seleccionar elementos a partir de la presencia de cadenas de texto dentro del valor del atributo:

[attr^=value]
Se aplica a elementos con un atributo "attr" cuyo valor comienza por "value" (el valor entre comillas):
li[class^="box-"]

[attr$=value]
Se aplica a elementos con un atributo "attr" cuyo valor termina en "value" (el valor entre comillas):
li[class$="-box"]

[attr*=value]
Se aplica a elementos con un atributo "attr" cuyo valor contiene "value" (el valor entre comillas) en cualquier lugar de la cadena:
li[class*="box"]

Distinguir (o no) mayúsculas de minúsculas en las búsquedas de valores

HTML distingue entre mayúsculas y minúsculas, así que a la hora de buscar valores de los atributos hay que tener esto en cuenta. Si queremos indicar a CSS que no distinga entre mayúsculas y minúsculas se puede utilizar el valor i antes del corchete de cierre, lo que hace que el navegador busque coincidencias independientemente de que sean mayúsculas o minúsculas. Para buscar:

li[class^="a" i]

Selecciona los elementos li que tengan una clase cuyo valor empieza por "a" independientemente de si está en mayúsculas o minúsculas.

Pseudo-clases

Las pseudo-clases son selectores que seleccionan elementos que tienen un estado específico, como que son el primer elemento de su tipo, o sobre los que está pasando el ratón. Actúan como si se hubiese aplicado una clase a una parte del documento. Las pseudo-clases empiezan con dos puntos : como por ejemplo :hover

article p:first-child {
   font-size: 120%;
   font-weight: bold;
}

Se puede escribir la pseudo-clase sin un selector de elemento antes. En el ejemplo anterior, podríamos escribir article :first-child y la regla se aplicaría a cualquer elemento que sea el primer hijo de un elemento <article> y no sólo al primer párrafo hijo como habíamos indicado

Pseudo-clases de acción de usuario

Algunas pseudo-clases sólo se aplican cuando el usuario interactúa de alguna forma con el documento. A estas pseudo-clases a veces se les llama pseudo-clases dinámicas. Ejemplos de estas pseudo-clases son :hover o :focus

Pseudo-elementos

Los pseudo-elementos se comportan de una forma similar a las pseudo-clases, sin embargo actúan como si se hubiese añadido un nuevo elemento HTML, el lugar de aplicar una clase a los elementos existentes. Los pseudo-elementos comienzan por dos puntos dobles :: como por ejemplo ::before

article p::first-line {
   font-size: 120%;
   font-weight: bold;
}

En este ejemplo es como si apareciese una etiqueta <span> seleccionando la primera línea de los párrafos dentro del elemento <article> y se adaptase automáticamente a los cambios de la línea. Afectaría a todos los párrafos dentro del elemento <article>, pero podemos combinarlo con una pseudo-clase de forma que sólo afecte al primer párrafo hijo de <article>: article p:first-child::fist-line

Generando contenido con ::before y ::after

Hay dos pseudo-elementos especiales que son utilizados conjuntamente con la propiedad content para insertar contenido en el documento utilizando CSS: los pseudo-elementos ::before y ::after

.box::before {
   content:"Esto se deberá mostrar antes que el otro contenido ";
}

El insertar texto desde CSS no es algo habitual, en cambio es más común utilizar estos pseudo-elementos para insertar un icono, para por ejemplo evitar que sea leído por un lector de pantalla

.box::after {
   content:" ➥ ";
}

Otro uso habitual es para añadir una cadena vacía a la que posteriormente se le da estilo como cualquier otro elemento de la página. En el siguiente ejemplo insertamos una cadena vacía que transformamos en un cuadrado al que damos un estilo:

.box::before {
   content:"";
   display:block;
   width: 100px;
   height: 100px;
   background-color: rebeccapurple;
   border: 1px solid black;
}

COMBINADORES CSS

Combinadores

Selector de descendientes

Es la combinación de distintos selectores con un espacio entre ellos. Estos selectores seleccionan elementos que son descendientes de otros selectores. En el ejemplo seleccionamos el elemento <p> que hay dentro de una clase .box

.box p{
   color:red;
}

Selector de combinación de elementos hijo

El selector de combinación de elementos hijo es un símbolo de "mayor que" > que selecciona sólo cuando los selectores identifican elementos que son hijos directos. Los elementos descendentes que se encuentren más abajo en la jeraquía no se seleccionan. Por ejemplo, para seleccionar los elementos <p> que son hijos directos de elementos <article>

article > p{
   color:red;
}

Hermanos adyacentes

El selector de elementos hermanos adyacentes es un símbolo de suma+ que selecciona un elemento que se encuentra justo al lado de otro elemento en el mismo nivel de la jerarquía. En el ejemplo, para seleccionar todos los elementos <img> que aparecen justo después de elementos <p>

p + img{
   color:red;
}

Un uso muy común es modificar el párrafo que va justo después del título.

Hermanos generales

El selector de combinación de elementos hermanos, incluso si no son directamente adyacentes, es el símbolo ~ En el ejemplo, para seleccionar todos los elementos <img> que aparecen después de elementos <p>

p ~ img{
   color:red;
}

Uso de selectores de combinación.

Los selectores de combinación se pueden usar con cualquiera de los selectores vistos anteriormente, aunque hay que tener cuidado de no crear selecciones muy específicas que tengan poca reutilización a lo largo del documento. En este caso es mejor crear una clase sencilla y aplicarla a los elementos en cuestión.

En el ejemplo, para seleccionar elementos de una lista con una clase "a" que son hijos directos de <ul>

ul > li[class="a"]{
   color:red;
}
